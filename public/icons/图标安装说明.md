# 🚨 图标文件缺失

## 重要提示

扩展**必须**添加图标文件才能在浏览器中加载！

### 必需文件：
- `icon16.png` (16x16 像素)
- `icon48.png` (48x48 像素)
- `icon128.png` (128x128 像素)

---

## 快速解决方案

### 方法 1：在线生成（推荐，最快）

1. 访问：https://www.favicon-generator.org/ 或 https://realfavicongenerator.net/
2. 上传图片或使用文字"译"生成
3. 下载 PNG 文件
4. 重命名为 icon16.png, icon48.png, icon128.png
5. 放入此目录：`public/icons/`
6. 重新构建：`npm run build`

### 方法 2：打开浏览器生成

1. 用浏览器打开 `create-icons.html`
2. 右键点击每个图标 → 图片另存为
3. 依次保存为 icon16.png, icon48.png, icon128.png
4. 放入此目录
5. 重新构建

### 方法 3：临时占位图（快速测试）

如果只想快速测试功能：

```powershell
# 下载任意小图片作为临时图标
# 或复制任意 PNG 文件三份，重命名为：
# icon16.png, icon48.png, icon128.png
```

## 设计建议

**风格：**
- 渐变背景（如 #667eea → #764ba2）
- 白色"译"字或"T"字母
- 圆角设计
- 简洁现代

**工具：**
- Figma、Canva（在线设计）
- Photoshop、GIMP（专业工具）
- Paint.NET（简单易用）

---

## 添加图标后

1. 确认文件名正确：icon16.png, icon48.png, icon128.png
2. 确认文件在 `public/icons/` 目录
3. 运行构建：`npm run build`
4. 在浏览器中重新加载扩展

✅ 图标添加成功后，扩展就可以正常使用了！
