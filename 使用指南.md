# 使用指南 - 翻译扩展

## 🎯 功能特性

### 1. 划词翻译（推荐）
这是最方便的功能！

**使用方法：**
1. 在任意网页上**选中**你想翻译的文字
2. 等待 1 秒，翻译浮窗自动出现在选中文字旁边
3. 浮窗显示翻译结果
4. 点击「📋 复制」按钮复制翻译结果
5. 点击「✕」关闭浮窗

**特点：**
- ✅ 自动检测语言
- ✅ 美观的渐变设计
- ✅ 智能定位（不遮挡文字）
- ✅ 流畅动画效果

### 2. 点击扩展图标翻译
手动输入文字翻译

**使用方法：**
1. 点击浏览器工具栏的扩展图标
2. 在弹窗中输入要翻译的文字
3. 选择源语言和目标语言
4. 点击「翻译」按钮
5. 查看并复制结果

### 3. 设置 API 密钥

**首次使用必须配置：**
1. 点击扩展图标
2. 点击右上角 ⚙️ 设置图标
3. 选择翻译服务商（火山翻译或微软翻译）
4. 填入 API 密钥
5. 点击「测试连接」验证
6. 点击「保存设置」

## 🔑 API 申请

### 火山翻译（推荐）
- **免费额度：** 200万字符/月
- **申请地址：** https://console.volcengine.com
- **申请步骤：**
  1. 注册并实名认证
  2. 开通「机器翻译」服务
  3. 创建访问密钥
  4. 复制 Access Key ID 和 Secret Access Key

### 微软翻译
- **免费额度：** 200万字符/月
- **申请地址：** https://azure.microsoft.com
- **申请步骤：**
  1. 创建 Azure 账号
  2. 创建「Translator」资源
  3. 选择免费层 F0
  4. 获取 API Key 和 Region

## ⚡ 快捷键（可自定义）

- `Ctrl+Q` - 翻译选中文本
- 点击扩展图标 - 打开翻译弹窗

## 💡 使用技巧

1. **选中文字后稍等**：浮窗会在 1 秒后自动显示
2. **调整浮窗位置**：如果浮窗遮挡内容，点击关闭后重新选择
3. **批量翻译**：可以选中多段文字，一次性翻译
4. **离线缓存**：相同的翻译会被缓存，第二次查询更快

## 🐛 常见问题

### Q: 选中文字后没有显示翻译？
A: 检查：
1. 是否已配置 API 密钥
2. 等待 1 秒让浮窗出现
3. 检查控制台是否有错误（F12）

### Q: 提示 API 错误？
A: 
1. 检查 API 密钥是否正确
2. 检查网络连接
3. 确认未超出免费额度

### Q: 浮窗位置不合适？
A: 
- 浮窗会自动调整位置避免超出屏幕
- 可以关闭后重新选择文字
